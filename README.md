# メモアプリ

## 説明
自分用のメモを保存、編集、削除し、管理できるアプリです。

## 使い方

1. ローカルに保存

    ``$ git clone https://github.com/m-ito66/memo-app.git``

2. ディレクトリに移動

    ``$ cd memo_app``

3. gemのインストール

    ``$ bundle install``

4. アプリの起動

    ``$ ruby memo_app.rb``

5. ローカル環境でメモアプリを利用できます。

    http://localhost:4567/memos
